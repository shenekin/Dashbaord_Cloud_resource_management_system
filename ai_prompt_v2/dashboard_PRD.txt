## 1️⃣ Dashboard 总体结构

### **顶部导航 / 全局**

* **全局搜索**：按用户、项目、资源、任务快速定位
* **全局状态**：

  * 系统健康：网关、核心服务是否可用
  * 告警总览：未处理告警数量
  * 今日任务/审批概览

### **左侧菜单 / 服务分区**

* 网关 & 安全
* 用户 & 权限
* 项目 & 资源
* ECS & 资源操作
* 监控 & 告警
* 自动化任务
* 财务 & 成本
* 审计 & 日志
* 消息通知



## 2️⃣ 各服务对应 Dashboard 设计

| 服务名称                     | Dashboard 核心模块 | 核心指标/功能                                        | 展示形式 / 操作入口                   |
| ------------------------ | -------------- | ---------------------------------------------- | ----------------------------- |
| **gateway-service**      | 网关概览           | - 请求总量/秒<br>- 成功率、错误率<br>- 响应时间分布<br>- 限流/熔断状态 | 图表：折线图、柱状图<br>操作：调整路由/限流策略    |
| **auth-service**         | 用户权限管理         | - 活跃用户数/总用户数<br>- 角色分布<br>- Token 使用情况/过期提醒    | 列表/饼图<br>操作：创建/修改用户/角色/权限     |
| **project-service**      | 项目管理           | - 项目/部门列表<br>- 各项目资源使用情况<br>- 多租户状态            | 树状列表/表格<br>操作：创建/删除/分配资源      |
| **ecs-service**          | ECS 管理         | - ECS 实例数量/状态<br>- CPU/内存/网络监控<br>- 快照、开关机操作历史 | 仪表盘/表格/折线图<br>操作：启动/停止/快照     |
| **resource-service**     | 公共资源           | - 镜像、VPC、弹性IP、模板使用情况<br>- 安全组数量及状态             | 表格/标签云<br>操作：创建/删除/绑定资源       |
| **quota-service**        | 配额管理           | - 各项目/用户资源使用 vs 配额<br>- 审批请求状态                 | 表格/饼图<br>操作：审批/调整配额           |
| **task-service**         | 自动化任务          | - 待执行任务列表<br>- 任务成功率/失败率<br>- 定时任务计划           | 表格/甘特图<br>操作：手动执行/暂停/编辑任务     |
| **monitor-service**      | 监控告警           | - 系统指标（CPU、内存、网络、磁盘）<br>- 告警规则触发数<br>- 告警处理状态  | 折线图/柱状图/告警列表<br>操作：确认/忽略/创建规则 |
| **billing-service**      | 成本统计           | - 本月/本季度费用<br>- 费用预测<br>- 各项目成本占比              | 柱状图/饼图/折线图<br>操作：导出报表         |
| **audit-service**        | 操作审计           | - 最近操作日志<br>- 高风险操作警告                          | 表格/搜索筛选<br>操作：导出日志/查看详情       |
| **notification-service** | 消息中心           | - 消息发送状态统计（成功/失败）<br>- 最近通知记录                  | 表格/折线图<br>操作：发送测试消息/查看日志      |



## 3️⃣ Dashboard 核心功能建议

1. **统一权限控制**

   * Dashboard 按 RBAC 显示模块（auth-service 提供权限）
   * 不同用户只看到允许的服务和操作

2. **多维度搜索与筛选**

   * 按项目、用户、服务类型、时间区间

3. **告警与通知整合**

   * 告警集中展示，支持点击跳转到具体服务/资源

4. **可操作性**

   * 例如 ECS 停机/启动、任务手动触发、审批通过/拒绝

5. **可视化与趋势分析**

   * 趋势图、饼图、热力图展示资源使用、成本分布、任务成功率

6. **导出 & 报表**

   * CSV/Excel/PDF 导出数据、报表定时生成

## 4️⃣ 技术建议

* **前端框架**：React + Element Plus
* **图表库**：ECharts
* **实时更新**：WebSocket
* **后端接口**：gateway-service 聚合各服务数据
